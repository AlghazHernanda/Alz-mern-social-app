## This website was built using MERN stack and other, namely:
- **MongoDB Database**
   - NoSQL Database for storing data
- **Express.js** 
- **React.js** 
- **Node.js**
   - Node Module : helmet.js, multer.js, nodemon.js, bcrypt.js, morgan.js, dotenv.js, path.js, cors.js
- **Axios**
- **React Hooks**
- **Context API**
- **REST API**
