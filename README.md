- **MongoDB Database**
- **Express.js** 
- **React.js** 
- **Node.js**

   - Node Module : multer.js
   - Node Module : nodemon.js
   - Node Module : bcrypt.js
   - Node Module : morgan.js
   - Node Module : dotenv.js
   - Node Module : path.js
   - Node Module : cors.js
- **Axios**
- **React Hooks**
- **Context API**
- **REST API**
